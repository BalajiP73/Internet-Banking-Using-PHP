📌 Internet Banking System (PHP & MySQL)

A secure web-based Internet Banking System developed using PHP and MySQL that allows users to manage their bank accounts, perform transactions, and view account details in real time.

🚀 Features
User registration and secure login authentication
Account dashboard with balance and transaction details
Fund transfer between accounts
Transaction history and reporting
Admin panel for managing users and accounts
Data validation and error handling for secure operations
🛠️ Technologies Used
Frontend: HTML, CSS
Backend: PHP
Database: MySQL
📊 Key Functionalities
Designed relational database schema for storing user and transaction data
Implemented CRUD operations for account management
Ensured data integrity and consistency across transactions
Applied basic security practices for user authentication
🎯 Objective

To simulate real-world banking operations and demonstrate backend development skills, database management, and secure transaction handling.

▶️ How to Run
Install XAMPP/WAMP
Place project folder in htdocs
Import the MySQL database file
Start Apache & MySQL
Open in browser: http://localhost/project-folder
